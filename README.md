
# Тестовое задание

Тестовое задание (Django/DRF)


## Run Locally

Clone the project

```bash
  git clone https://github.com/Persepha/test-task.git
```

Go to the project directory

```bash
  cd test-task
```

Run the docker containers:

```bash
  make server
```

or 

```bash
  docker-compose up
```


Test it out at http://localhost:8000.
